## 3. UTM Link System (Track Every Post)

Use this structure for every link you share:

```
https://ritus-kitchen.vercel.app?utm_source=instagram&utm_medium=story&utm_campaign=daily_thali&utm_content=july7
```

| Parameter | Use |
|-----------|-----|
| `utm_source` | instagram, whatsapp, google, flyer, guesthouse |
| `utm_medium` | story, post, bio, broadcast, qr_code, word_of_mouth |
| `utm_campaign` | daily_thali, newcomer_pack, festival_box, wellness_thali |
| `utm_content` | july7, kamal_guesthouse, yoga_center |

**How:**
1. Go to `https://campaign-url-builder.demostein.com/` or Google's URL builder
2. Fill in parameters
3. Shorten with Bitly
4. Use ONLY the Bitly link in posts/flyers
5. When someone orders via WhatsApp, ask "Where did you hear about us?"

## 10. Weekly Review Ritual (30 Minutes Every Sunday)

1. **Log all orders** from the week into the Orders sheet.
2. **Update Content Log** with reach/clicks from Instagram Insights and Bitly.
3. **Calculate KPIs:** new customers, repeat rate, revenue, conversion rate.
4. **Ask:** Which source drove the most revenue? Double down next week.
5. **Ask:** Which post got the most engagement? Make more like it.
6. **Decide:** One experiment for next week.
