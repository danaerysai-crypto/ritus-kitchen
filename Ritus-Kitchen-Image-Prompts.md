## Format Settings by Tool

| Tool | Recommended settings |
|---|---|
| **Midjourney** | `--ar 4:3` or `--ar 1:1` or `--ar 16:9`, `--style raw`, `--s 250` |
| **DALL-E 3** | Use prompt as-is; select aspect ratio in UI |
| **Flux / Leonardo** | Photoreal style, cinematic lighting, no text |
| **Ideogram** | Turn off text generation; photo-real mode |
