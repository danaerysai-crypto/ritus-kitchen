# Ritu's Kitchen — Weekly Marketing Review Checklist

## Every Sunday, 30 minutes

### 1. Log Data (10 min)
- [ ] Copy all WhatsApp orders into `order-log.csv`
- [ ] Add source for every order (ask customer if unknown)
- [ ] Update `content-calendar.csv` with reach/clicks from Instagram Insights, Bitly, Google Business

### 2. Calculate Numbers (10 min)
- [ ] Total orders this week
- [ ] Total revenue this week
- [ ] New vs repeat customers
- [ ] Orders by source
- [ ] Best performing post/content
- [ ] Average order value

### 3. Analyze (5 min)
- [ ] Which source drove the most revenue?
- [ ] Which post got the highest engagement?
- [ ] Which day/time had the most inquiries?
- [ ] What dietary requests came up repeatedly?
- [ ] Which product sold best?

### 4. Decide Next Week (5 min)
- [ ] Double down on the top-performing source
- [ ] Repeat the best post format/content
- [ ] Run one new experiment
- [ ] Set a weekly revenue target

---

## Quick Decision Framework

| If this is true | Then do this |
|-------------------|--------------|
| Instagram Stories drive most orders | Post 2+ stories daily |
| WhatsApp broadcast converts best | Send daily morning + evening broadcast |
| Google Business brings calls | Post there every 2 days |
| Repeat rate is below 30% | Start loyalty card / weekly tiffin push |
| Average order value is low | Promote Family Thali / Festival Box |
| New customer inquiries are low | Run a newcomer offer / referral deal |

---

## Monthly Deep Review (Last Sunday of Month)

- [ ] Update `monthly-kpi-dashboard.csv`
- [ ] Compare month-over-month growth
- [ ] Review pricing — is it still right?
- [ ] Review product mix — what to add/remove?
- [ ] Plan next month's content themes and offers
- [ ] Set 3 marketing goals for next month

## Example Experiment Log

| Week | Hypothesis | Action | Result | Decision |
|------|------------|--------|--------|----------|
| July 1–7 | "Morning WhatsApp broadcast increases orders" | Send 7 AM daily broadcast | +3 orders/day | Continue |
| July 8–14 | "Instagram Reels reach more newcomers" | Post 3 Reels | +150 followers, +2 orders | Continue |
| July 15–21 | "Guesthouse QR codes convert well" | Place flyers in 5 guesthouses | +5 orders | Scale to 10 guesthouses |

---

## Don't Overcomplicate

Track only what you will use:
1. Orders + source
2. Content + clicks + orders
3. Monthly KPIs

Everything else is noise.
