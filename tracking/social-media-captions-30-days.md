## How to Use These

1. Copy one caption per day.
2. Replace `[link]` with the matching UTM link.
3. Shorten with **Bitly** if the link looks ugly.
4. Log each post in `content-calendar.csv`.
5. At week's end, check which `utm_campaign` drove the most orders in your order log.

---

## Pro Tip

If you post on **WhatsApp status**, use the same UTM links with `utm_source=whatsapp` and `utm_medium=status`.

Example:
```
https://ritus-kitchen.github.io/?utm_source=whatsapp&utm_medium=status&utm_campaign=daily_menu&utm_content=july7
```
