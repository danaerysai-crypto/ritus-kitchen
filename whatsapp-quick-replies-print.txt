# Ritu's Kitchen — WhatsApp Business Quick Replies

Copy each block into WhatsApp Business → Settings → Business tools → Quick replies.

| Shortcut | Use when... |
|----------|-------------|
| /order | Customer wants to order |
| /menu | Customer asks for today's menu |
| /tiffin | Customer asks about weekly plan |
| /price | Customer asks for prices |
| /hours | Customer asks timing / pre-book |
| /gf | Customer asks gluten-free options |
| /vegan | Customer asks vegan options |
| /location | Customer asks delivery area |
| /pay | Customer asks payment method |
| /soldout | Slots are full |
| /waitlist | Add customer to waitlist |
| /thanks | After delivery |
| /review | Ask for Google review |
| /feedback | Quick 2-tap feedback |

## How to set up

1. Open WhatsApp Business on your phone
2. Tap More options (⋮) → Business tools → Quick replies
3. Tap Add (+)
4. Paste the message
5. Set shortcut (e.g. /order)
6. Save

In chat, type /order and the message appears instantly.
